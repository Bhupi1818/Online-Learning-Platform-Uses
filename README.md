# Online Learning Platform Usage Analysis
---------------------------------------------------------------------------------------------------------------------------------
This repository contains a comprehensive data analysis of how different demographics interact with online learning platforms. Using Python and Jupyter Notebooks, this project explores trends in platform preference, completion rates, and user satisfaction.

---------------------------------------------------------------------------------------------------------------------------------

# Project Overview
The goal of this project is to understand the behavior of learners across various digital education platforms. The analysis covers:
* Demographics: How age groups differ in their learning purposes.
* Platform Performance: Comparing satisfaction scores and completion rates across major providers.
* Engagement: Analyzing average hours spent per week and preferred content types (Video, Text, Interactive).
* Device Usage: Identifying the primary hardware used for digital education.

---------------------------------------------------------------------------------------------------------------------------------

# Repository Structure
* online_learning_platform_usage_dataset.csv: The raw dataset used for the analysis.
* Online_Learning_Platform.ipynb: The main Jupyter Notebook containing data cleaning, processing, and visualization code.
* Visualizations(.png): High-resolution charts generated from the data.

---------------------------------------------------------------------------------------------------------------------------------

# Key Insights
* Primary Platforms: A comparison of market share between top e-learning providers.
* Completion Rates: Insights into which platforms successfully keep students engaged until the end of a course.
* User Roles: Analysis of work hours and learning habits based on professional roles (Student, Professional, etc.).

---------------------------------------------------------------------------------------------------------------------------------

# Tools Used
* Language: Python
* Libraries: Pandas, Matplotlib(for data visualization)
* Environment: Jupyter Notebook

---------------------------------------------------------------------------------------------------------------------------------

# Conclusion

As the landscape of digital education continues to evolve, understanding user behavior is key to building better learning experiences. This project serves as a foundational step in identifying what truly drives student success in a virtual environment.
